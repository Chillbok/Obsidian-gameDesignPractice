## 개조부품의 분류

개조부품에 대한 자세한 명세는 추후 **개조부품 기획**에서 진행하도록 한다.
아래는 개조부품 종류에 대한 분류이다.

| 개조부품 유형                                              | 설명                                                                                                                                                                                                                      |
| ---------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 방어구 능력치 개조부품                                         | 방어구 능력치를 추가로 제공할 수 있는 방어구 개조부품.<br>특정 방어구 능력치에 **+5점을 제공**하는, 에너지 수용량이 1인 개조부품과, **+10점을 제공**하는 에너지 수용량이 3인 개조부품이 있다.<br>각 방어구 당 하나의 슬롯에만 장착할 수 있다.                                                                     |
| 무기 개선 개조부품                                           | 무기의 재장전 속도나 무기 조작 속도^[재장전 속도를 제외한 무기 조작 속도. 예를 들면 조준 속도, 무기 전환 속도 등이 포함됨.], 피격 반동 감소 등 무기를 사용할 때                                                                                                                        |
| 능력 충전 개조부품                                           | 능력 충전 효과를 부여하는 개조부품.<br>예를 들어, 한 능력을 사용했을 때 다른 능력 에너지를 소량 회복하는 등의 효과를 부여한다.                                                                                                                                             |
| 힘의 보주 생성 개조부품                                        | 힘의 보주를 생성하는 수단을 늘려주는 개조부품들이다.                                                                                                                                                                                           |
| 힘의 보주 효과 개조부품                                        | 힘의 보주를 획득했을 때 효과들을 추가하는 개조부품.<br>생명력을 회복시키거나, 능력 에너지를 소량 회복시키는 것이 가능함.                                                                                                                                                  |
| 방어구 충전^[힘의 보주 획득 시 기본적으로 1개씩, 최대 3개까지 획득 가능] 생성 개조부품 | 방어구 충전을 획득하는 수단을 늘려주는 개조부품.                                                                                                                                                                                             |
| 방어구 충전 사용 개조부품                                       | 충전된 방어구 충전을 소모해 플레이어에게 이득이 되는 효과를 얻을 수 있도록 하는 개조부품이다.^[마나를 소모하는 패시브 스킬을 플레이어에게 추가하되, 소모하는 마나를 '방어구 충전'이라고 생각하면 이해하기 쉬울 것이다.]<br>방어구 충전을 n개 소모하여 작동하는 효과, 방어구 충전 하나 당 15시간의 지속시간을 가지고 지속시간동안 버프를 부여하는 효과 이렇게 두 가지가 존재한다. |
| 탄약 탐지 개조부품                                           | 적 처치 시 탄약을 생성하는 빈도수를 늘려주는 개조부품.                                                                                                                                                                                         |

## 방어구 부위별 장착 가능한 개조부품

| 방어구 부위 | 장착 가능한 개조부품                                                                                                                                                      |
| ------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 머리     | 탄약 생성 능력 개선, 궁극기 성능 개선, 무기 조준 성능 개선과 관련된 개조부품들이 있다.                                                                                                              |
| 팔      | 무기 조작성 및 재장전 속도에 관여하는 개조부품, 수류탄/근접 공격 성능에 영향을 끼치는 개조부품들이 있다.                                                                                                     |
| 몸통     | 특정 피해를 감소시키는 개조부품들, 검 무기 관련 개조부품, 피격 반동 개선 개조부품들, 생명력 능력치 증가 개조부품 등이 있다.                                                                                         |
| 다리     | 적 처치 시 드롭되는 탄약에서 더 많은 탄약을 얻게 해주는 개조부품, 무기 능력치 증가 개조부품, 힘의 보주 획득 효과를 제공하는 개조부품이 있다.<br>또한 플레이어의 기본 이동 속도를 늘려주는 개조부품도 존재한다.                                        |
| 직업     | 필살기^[일정 체력 아래로 떨어진 PVE 적에게 사용할 수 있는 처형 모션. 처형 애니메이션이 끝나면 적은 반드시 사망한다.]를 사용했을 때 방어구 충전을 사용해서 효과를 제공하거나, 방어구 충전을 제공하는 개조부품들이 모여 있다.<br>또는 직업 능력과 관련된 개조부품들이 모여 있다. |
